# CP2

## 프로젝트 개요
음성을 통해 감정(기쁨, 슬픔, 분노, 당황, 중립)을 분류하고 감정에 적합한 답변을 생성하여 음성으로 들려주는 챗봇 서비스
***
<br/><br/>

## 프로젝트 방법
### Voice Part
ResNet모델을 기반으로 소리의 높낮이(데시벨)와 소리의 크기(주파수)를 분류하는 모델을 각각 생성하여 ensembel 적용.
***
<br/><br/>

### Chat Bot Part
사전 학습 모델인 KoGPT-2를 통해 각 감정에 따른 데이터로 미세 조정을 진행. 
***
<br/><br/>

## 최종 코드 결과
<img width="522" alt="image" src="https://user-images.githubusercontent.com/94213374/195053698-4f53876b-f0a8-4526-818e-ade83fc7a868.png">
위의 그림과 같이 코드를 실행하면 5초동안 녹음을 할 수 있습니다.     
모델에 입력되기 위해 녹음되는 파일의 형식인 m4a를 wav 파일 형식으로 변환합니다.   

목소리를 통해 감정을 먼저 분류합니다.    
   
목소리를 텍스트로 (STT) 변환을 한 후 감정에 해당하는 챗봇 모델에 입력됩니다.   
   
챗봇 모델이 답변을 생성한 후 텍스트를 음성으로 (TTS)으로 변환하여 들려줍니다.
